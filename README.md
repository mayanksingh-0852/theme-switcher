# theme-switcher
A React-based Theme Switcher application built using Context API that allows users to toggle between light and dark modes seamlessly. It demonstrates global state management, component reusability, and dynamic UI updates using Tailwind CSS.


# 🌗 React Theme Switcher using Context API

## 🚀 Overview

This project is a **Theme Switcher Web App** built using **React Context API**.
It allows users to toggle between **Light Mode** and **Dark Mode** across the entire application.

---

## 🧠 What I Learned

* How to use **React Context API**
* Managing **global state**
* Using **useState & useEffect**
* Implementing **dark/light theme toggle**
* Styling using **Tailwind CSS**
* Creating **reusable components**

---
